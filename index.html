    Project Description

    JavaScript Quiz/ questionnaire application build for user data collection.

    With the advantage of the app being cost-effective, scalable and modified to be tailored for a specific data collection.

    Software Used

    Motivation

    To develope an Quiz / questionnaire application for user input collection. In situations whereby collecting user experience is possible.

    This quiz app can be applied, modified and or tailored to multiple situations.

    Real-World Applications

    1. User Feedback
    2. Targeted Surveys
    3. Scalable questionnaire
    4. Cost-Effective Data Collection

    Future features

    These are future features I want to implement.

    1. User can share the result of a quiz on social media.
    2. Add multiple quizzes to the application. User can select which one to take.
    3. User can create an account and have all the scores saved in his dashboard
